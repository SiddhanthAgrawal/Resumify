Let us guide you through the process...

  1. Run the code in the repl.it embed by clicking on    or open the code in repl.it by clicking on  . If you want to run this code in your own environment, you can download the files as a zip from Files tab (on the left side navigation bar).

  2. The program will ask you for various input to add to your resume.

  3. After going through the whole program, click on the files tab on the left side navigation bar and click on the file named "template.tex". Then go to overleaf.com and start a new blank project. Copy the text from template.tex and paste it in the overleaf editor. Below you can see how the overleaf editor should look like.

  1. In the overleaf editor you can compile the tex file and download the pdf of your resume. Having the tex code for the resume gives you the freedom to edit the resume later as you want.
